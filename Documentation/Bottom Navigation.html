An Android bottom navigation with customizable tabs.

Use the Touch button on the component to get information about the selected tab.

Layer Inputs

Enable

A boolean that is true when the tab bar is displayed.

Only Show Title on Selected Tab

A boolean that determines if titles should be only shown on the selected tab or on all tabs.

Number of Tabs

A number representing the number of items in the bottom navigation.

Switch to Tab

The index of the tab to switch to, starting at 0.

Switch Tab

A pulse that changes the currently selected tab.

Background

The background color of the bottom navigation.

Position Y

The vertical position of the tab bar relative to the bottom of its parent group or screen.

Opacity

The opacity of the tab bar.

Icon Color (Selected)

The color of the icon of the selected tab.

Title Color (Selected)

The color of the title of the selected tab.

Icon Color (Unselected)

The color of the icons of the unselected tabs.

Title Color (Unselected)

The color of the titles of the unselected tabs.

Layer Outputs

Selected Tab Index

An index that represents the currently selected tab, starting at 0.

Selected Tab Title

A text string that represents the title of the currently selected tab.
