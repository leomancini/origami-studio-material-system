An Android slider that can be either continuous or discrete.

Use the Touch button on the component to get information about the current progress or value.

Layer Inputs

Enable

A boolean that is true when the floating action button is displayed.

Bottom Margin

A number representing the relative distance from the bottom of the screen. Useful for offsetting the position of the floating action button when a system navigation bar is present.
